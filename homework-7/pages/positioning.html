Експерименти з позиціонуванням

position: fixed
(завжди на екрані)

Static, Relative та Absolute позиціонування

Static
Relative
top: 20px
left: 50px
Absolute
top: 50px
right: 50px

Static - нормальний потік. Relative - зміщений відносно себе. Absolute - відносно батьківського контейнера з position: relative.

position: sticky (прилипає при прокрутці)

Прокрутіть сторінку, щоб побачити різницю між fixed та sticky

Fixed - завжди залишається на тому самому місці екрана.

Sticky - поводиться як relative, доки не досягне заданої позиції, потім "прилипає" як fixed.
